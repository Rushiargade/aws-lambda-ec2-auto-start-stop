# AWS Lambda EC2 Auto Start/Stop

This project automates the start and stop of EC2 instances based on instance tags using AWS Lambda and Boto3.

## Tags Used
- Action = Auto-Stop
- Action = Auto-Start

## AWS Services Used
- AWS EC2
- AWS Lambda
- AWS IAM
- Amazon CloudWatch

## Functionality
- Stops EC2 instances tagged with `Auto-Stop`
- Starts EC2 instances tagged with `Auto-Start`
